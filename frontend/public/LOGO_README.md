# Logo Dosyası

Bu klasöre NTMYO (Naci Topçuoğlu Meslek Yüksekokulu) logosunu ekleyin.

## Logo Özellikleri

- **Dosya adı**: `logo.png` veya `logo.svg`
- **Konum**: `frontend/public/logo.png`
- **Önerilen boyut**: 
  - Sidebar için: 40x40px (minimum)
  - Login sayfası için: 64x64px
  - Ana sayfa için: 64x64px

## Logo Açıklaması

Logo şu özelliklere sahip olmalıdır:
- Koyu mavi daire (#0a294e)
- Kırmızı banner (#bf1e1d veya #dc2626)
- "NACI TOPÇUOĞLU MESLEK YÜKSEKOKULU" yazısı
- "NTMYO" kısaltması

## Kullanım

Logo otomatik olarak şu sayfalarda gösterilir:
- Sidebar (sol menü)
- Login sayfası
- Ana sayfa (dashboard)

Logo yüklenemezse otomatik olarak "NT" placeholder gösterilir.
