# Logo Dosyası Ekleme Talimatları

## Logo Dosyası Konumu
```
frontend/public/assets/ntmyo-logo.png
```

## Logo Özellikleri
- **Format:** PNG (şeffaf arka plan tercih edilir)
- **Boyut:** En az 200x200px (daha yüksek çözünürlük tercih edilir)
- **Dosya Adı:** `ntmyo-logo.png` (tam olarak bu isim)

## Logo Ekleme Yöntemleri

### Yöntem 1: Terminal ile
```bash
# Logo dosyanızı bu klasöre kopyalayın
cp [logo-dosyanızın-yolu] "/Users/enescikcik/Desktop/NTMYO MEDEK /frontend/public/assets/ntmyo-logo.png"
```

### Yöntem 2: Finder ile
1. Logo dosyanızı Finder'da bulun
2. `frontend/public/assets/` klasörünü açın
3. Dosyayı sürükleyip bırakın
4. Dosya adını `ntmyo-logo.png` olarak değiştirin

### Yöntem 3: Cursor ile
1. Logo dosyanızı Finder'da bulun
2. Cursor'da `frontend/public/assets/` klasörünü açın
3. Dosyayı sürükleyip bırakın
4. Dosya adını `ntmyo-logo.png` olarak değiştirin

## Logo Görüntülenecek Yerler
- ✅ Sidebar (sol menü)
- ✅ Login sayfası
- ✅ Diğer sayfalarda gerekirse

## Not
Logo dosyası eklenene kadar geçici bir fallback görünümü gösterilecektir (mavi gradient kutu içinde "NT" yazısı).
