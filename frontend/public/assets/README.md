# Logo Dosyası

Bu klasöre NTMYO logosunu `ntmyo-logo.png` adıyla ekleyin.

Logo dosyası şu özelliklere sahip olmalıdır:
- Format: PNG (şeffaf arka plan tercih edilir)
- Boyut: En az 200x200px (daha yüksek çözünürlük tercih edilir)
- Dosya adı: `ntmyo-logo.png`

Logo otomatik olarak şu yerlerde görüntülenecektir:
- Sidebar (sol menü)
- Login sayfası
- Diğer sayfalarda gerekirse

Logo yüklenene kadar varsayılan bir ikon gösterilecektir.
